# 📝 Laravel Blog Application

## 📌 Overview
A simple blog application built using Laravel framework following MVC architecture. It supports full CRUD functionality for blog posts.

---

## 🚀 Features
- Create blog posts
- Read/view posts
- Update/edit posts
- Delete posts
- MVC structure (Model-View-Controller)

---

## 🛠️ Tech Stack
- Laravel
- PHP
- MySQL
- Blade Templates

---

## ⚙️ Installation

```bash
git clone https://github.com/Mitu12/laravel-blog-app.git
cd laravel-blog-app
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
